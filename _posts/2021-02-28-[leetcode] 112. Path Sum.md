[leetcode 112번 문제](https://leetcode.com/problems/path-sum/)

문제 이해 
* targetSum와 일치하는 이진트리 경로의 합이 있는지 

```
class Solution:
    def hasPathSum(self, root: TreeNode, sum: int) -> bool:
        def binaryTreePaths(root):
            if root is None: 
                return []
            if (root.left == None and root.right == None):
                return [root.val]

            left_subtree = binaryTreePaths(root.left)  
            right_subtree = binaryTreePaths(root.right)
            full_subtree = left_subtree + right_subtree  

    
            list1 = []
            for leaf in full_subtree:  # middle part of the comprehension
                list1.append(root.val + int(leaf))  # the left part 

            return list1
        
        if sum in binaryTreePaths(root):
            return True
        else:
            return False
```
